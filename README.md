
# Analysis of Online Education Information Using Big Data

## Overview
This project analyzes online education activity data using Python, Pandas, and Matplotlib. 
The system helps identify:
- Student engagement patterns
- Most popular courses
- Peak learning hours
- User activity trends

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Data preprocessing
- Activity analysis
- Course popularity tracking
- Visualization dashboards
- User engagement analytics

## Project Structure
```

online_education_bigdata_project/
│
├── data/
├── notebooks/
├── src/
├── visualizations/
├── docs/
├── requirements.txt
├── README.md
└── .gitignore

```

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
```

## Output
- Bar chart of popular courses
- Activity distribution by hour
- Distinct user analytics
